# Design System

## Colors

### Primary Colors

- **Primary**: `#0352A3` - Use class: `bg-primary`, `text-primary`, `border-primary`
- **Primary Light**: `#0070E5` - Use class: `bg-primary-light`, `text-primary-light`
- **Primary Lighter**: `#3EAAFF` - Use class: `bg-primary-lighter`, `text-primary-lighter`

### Neutral Colors

- **White**: `#FFFFFF` - Use class: `bg-white`, `text-white`
- **Grey Text**: `#55779A` - Use class: `text-grey-text`
- **Grey Light**: `#D9DFEB` - Use class: `bg-grey-light`, `border-grey-light`

### Background Colors

- **Light Blue Extra**: `#DFEAFF` - Use class: `bg-light-blue-extra`
- **BG Grey Light**: `#F6F9FF` - Use class: `bg-grey-light`

## Fonts

### Font Families

- **Inter**: Default sans-serif font - Use class: `font-sans` or `font-inter`
- **Instrument Serif**: Serif font for headings - Use class: `font-serif` or `font-instrument-serif`

### Usage

```tsx
// Default (Inter)
<p className="font-sans">Regular text</p>

// Instrument Serif
<h1 className="font-serif">Heading text</h1>
```
